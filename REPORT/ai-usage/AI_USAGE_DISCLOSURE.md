# Uso de IA

Use IA libremente como permite el reto, principalmente para redaccion, revision, organizacion de fases, prompts de trabajo, chequeos de consistencia y apoyo al debugging.

Mantengo como propias las decisiones de alcance, arquitectura, separacion de autoridad, criterios de readiness, trazabilidad por fases, seleccion final de modelo y responsabilidad del entregable.
