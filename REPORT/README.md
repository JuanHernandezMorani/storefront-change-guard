# Reportes

Esta carpeta contiene registros de ejecucion, prompts, changelog operativo y disclosure de uso de IA. Preservo los originales completos en `.original_en/REPORT/`.

## Lectura final

- `executions/` resume validaciones y gates.
- `prompts/` registra instrucciones usadas durante el desarrollo.
- `ai-usage/` explica como use IA.
- `changelog/` mantiene decisiones y cambios operativos.
