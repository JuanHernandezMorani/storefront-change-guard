# AI Usage Disclosure

AI-assisted tools were used during the development of this prototype for tasks such as:

- Exploring implementation alternatives.
- Drafting initial code and documentation.
- Reviewing edge cases.
- Identifying potential test scenarios.
- Improving clarity and consistency in technical writing.

## Author-owned work

The following aspects were defined, selected, integrated, reviewed, tested, and owned by the author:

- Prototype scope and prioritization.
- Architecture and trust boundaries.
- Local-first inference strategy.
- Acceptance criteria and readiness policies.
- Controlled e-commerce regression scenarios.
- Validation workflow using isolated Git worktrees.
- Integration decisions.
- Final code review, debugging, tests, and delivery.

AI-generated suggestions are treated as drafts. They are reviewed, adapted, tested, or rejected according to project requirements.

## Record policy

Prompt records are retained only when they materially affect design, implementation, testing, or documentation. Records should summarize purpose and outcome without retaining secrets or unnecessary raw context.
