# Auditoria

Esta carpeta contiene la trazabilidad de fases, fixes, enhancements y reviews. Mantengo los originales en ingles dentro de `.original_en/AUDIT/` y dejo esta entrega en espanol para la exposicion.

## Convencion

- `Phase-NN`: fase de implementacion.
- `Phase-NN-FIX-NN`: correccion asociada a la fase donde nacio el problema.
- `Phase-NN-ENHANCE-NN`: mejora visual, documental o de presentacion asociada a su fase de origen.
- `Phase-NN-REVIEW-NN`: review independiente que verifica o rechaza una fase, fix o enhance.
